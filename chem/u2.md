# 2.7 VESPER
`倒推方法`: lone pairs因为没有外面的原子拉,所以更近,角度更大 

1. 2段
    - Linear 180deg
2. 3段
    - Trignoal Planar 120deg
    - Bent (Trignoal Planar退化, 根据`倒推方法` 造成 < 120deg)
3. 4段
    - Tetra Hedral 109.5deg
    - Trigonal Pyramidal (Tetra Hedral退化 <109.5)
    - Bent (Trigonal Pyramidal退化 <109.5)
4. 5段
    - Trigonal Bipyramidal(横着Trigonal Planar + 纵Linear 角度不相同但组成的角度相同于横纵两个部分带的角度)
    退化: Seesaw -> T-Shaped -> Linear

5. 6段
    - Octahedral(90度 均匀)
    退化: Square Pyrimidal -> Square Planar
